'1' : 캐릭터 일인칭 시점
'2' : 캐릭터의 뒤에서 보는 시점
'3' : 맵 상단에서 보는 시점
'4’ : 와이어프레임 모드로 전환
'5’ : 플랫 셰이딩 모드로 전환


방향키 상 : 캐릭터가 현재 보고 있는 방향으로 전진
방향키 좌 : 캐릭터를 시계반대방향으로 회전
방향키 우 : 캐릭터를 시계방향으로 회전

차로에서 나오는 자동차들을 피해서, 맵의 끝 지점까지 가세요.
